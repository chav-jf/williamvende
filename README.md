# WILLIAM VENDE

Sitio web de **WILLIAM VENDE** — venta de productos Apple originales importados (AirPods Pro y Apple Watch)
al detal y por mayor en Pasto, Nariño 🇨🇴.

Hero interactivo (video de fondo con scrubbing + efecto máquina de escribir), catálogo con carrito que arma
el pedido por **WhatsApp**, reseñas, y sección de **TikTok** (@william.vende7).

## Stack
- Vite + React 19
- Tailwind CSS v4
- Motion (`motion/react`)
- lucide-react

## Desarrollo
```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # build de producción en /dist
```
